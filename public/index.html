🙊 Système de Confessions

Permettez aux membres d'envoyer des confessions anonymes via /confess. Seul le salon d'entrée est requis pour limiter l'usage de la commande.

Salons autorisés

La confession est publiée anonymement dans le salon où la commande est tapée. Limitez ici les salons autorisés.

Si aucun salon n'est ajouté, la commande est utilisable partout.

Apparence

💡 Comment ça marche : Un membre tape /confess dans le salon configuré. Un formulaire s'ouvre. Sa confession est publiée anonymement dans le salon de publication sous la forme Confession anonyme #N. Si un salon modération est défini, les mods reçoivent un log privé avec le nom de l'auteur.
